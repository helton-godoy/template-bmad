# Criar Wireframe - Lista de Verificação de Validação

## Estrutura de Disposição

- [ ] Dimensões do ecrã adequadas para o tipo de dispositivo
- [ ] Alinhamento da grade (20px) mantido
- [ ] Espaçamento consistente entre elementos de IU
Hierarquia adequada (cabeçalho, conteúdo, rodapé)

## Elementos UI

- [ ] Todos os elementos interativos claramente marcados
- [ ] Botões, entradas e controles devidamente dimensionados
- [ ] Etiquetas de texto legíveis e adequadamente dimensionadas
- [ ] Elementos de navegação claramente indicados

## Fidelidade

- [ ] Jogos solicitados nível de fidelidade (baixo/médio/alto)
- [ ] Nível de pormenor adequado
- [ ] Conteúdo do lugar utilizado quando necessário
- [ ] Sem decoração desnecessária para baixa fidelidade

Anotações

- Interações-chave anotadas
- [ ] Indicadores de fluxo presentes se multi-tela
- [ ] Notas importantes incluídas
- Os propósitos do elemento são claros.

## Qualidade técnica

- [ ] Todos os elementos devidamente agrupados
- [ ] Elementos de texto têm recipienteId
- Apertado na grelha
- [ ] Sem elementos com `isDeleted: true`
- O JSON é válido.
- [ ] Arquivo salvo para corrigir a localização
