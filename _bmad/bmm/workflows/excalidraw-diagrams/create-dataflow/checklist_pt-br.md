# Criar Diagrama de Fluxo de Dados - Lista de Verificação de Validação

## Nota DFD

- [ ] Processos mostrados como círculos/espíritos
- [ ] Lojas de dados mostradas como linhas paralelas ou retângulos
- [ ] Entidades externas mostradas como retângulos
- [ ] Fluxos de dados mostrados como setas marcadas
- [ ] Segue a notação padrão DFD

## Estrutura

- [ ] Todos os processos numerados corretamente
- [ ] Todos os fluxos de dados marcados com nomes de dados
- [ ] Todas as lojas de dados nomeadas apropriadamente
- [ ] Entidades externas claramente identificadas

## Completo

- [ ] Todas as entradas e saídas contabilizadas
- [ ] Nenhum processo órfão (desligado)
- [ ] Conservação dos dados mantida
- [ ] Nível adequado (contexto/nível 0/nível 1)

## Disposição

- [ ] Direção do fluxo lógico (da esquerda para a direita, de cima para baixo)
- [ ] Sem fluxos de dados de cruzamento onde evitável
- [ ] Disposição equilibrada
- [ ] Alinhamento da grade mantida

## Qualidade técnica

- [ ] Todos os elementos devidamente agrupados
- [ ] As setas têm ligações adequadas
- [ ] Texto legível e devidamente dimensionado
- [ ] Sem elementos com `isDeleted: true`
- O JSON é válido.
- [ ] Arquivo salvo para corrigir a localização
